# Simple E-commerce Checkout

A basic Flutter app demonstrating a simple e-commerce checkout form with shipping address and payment details.

## Features

- Shipping Address Form
- Payment Details Form
- Input Validation
- Order Summary

## Dependencies

- provider: For state management.
- intl: For formatting currencies and dates.

## Getting Started

1.  Clone the repository.
2.  Run `flutter pub get`.
3.  Run `flutter run`.
